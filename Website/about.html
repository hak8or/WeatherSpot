Windows Anwendung

WeatherSpot App is a client type application which connects to weatherspot.us website to fetch and display the data on a graph. The data itself is gathered by embedded devices which monitor and record into Influx Database properties like temperature, humidity or pressure. In addition, the application includes console feature which allows manual querries into the database. The elementary statistics section was added for the convinience of a user. This program uses two open source libraries JSON.NET for deserialization of JSON data and Dynamic Data Display for graphs plotting.
